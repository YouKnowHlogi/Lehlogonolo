Web App Development

These web apps showcase a combination of front-end development skills, including JavaScript for interactivity, HTML/CSS for structure and design, and additional technologies specific to each application (e.g audio handling for Play Drums). These web apps will demonstrate your proficiency in creating engaging and interactive web experiences.

The Simon Game is a classic memory-based electronic game brought to life on the web. Players must follow a sequence of colored lights and sounds produced by the game and then replicate the sequence to advance to the next level.

Skills Showcased:

JavaScript: Used for handling game logic and user interactions.
HTML/CSS: Responsible for structuring the page and styling the user interface.
DOM Manipulation: Utilized to dynamically update the game state on the webpage.
Event Handling: Captures and responds to user input for an engaging experience.

The Dice Game is a digital rendition of the classic dice-rolling experience. Players can roll virtual dice and experience the thrill of chance in a fun and interactive way.

Skills Showcased:

JavaScript: Implements the randomization of dice values and manages game rules.
HTML/CSS: Structures the visual representation of the dice and provides an appealing layout.
Animation: Enhances user experience through smooth transitions and visual effects.
Responsive Design: Ensures the game adapts to various screen sizes for a seamless user experience.

The Drum Kit web app is a virtual drum set that allows users to create beats and rhythms by interacting with different drum elements. It provides an immersive musical experience right in the browser.

Skills Showcased:

JavaScript: Enables drum sounds on user interaction and handles real-time events.
HTML/CSS: Structures the drum kit components and provides an intuitive user interface.
Audio API: Integrates sound files and manages playback for each drum element.
